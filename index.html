Sketch Premise Generator and Other Writing Tools

This is Estevan Galvez's extremely basic GitHub site.
Here you can find some simple web applets I've made.

My Sketch Premise Generator is available here. It generates silly ideas. A more accurate name would be "Premise Generator" since you can use it for pretty much anything, but I don't feel like changing the name so long after the original release.

Here's a simpler word-pairing / word-grouping applet called Word Group Generator.

Here's a less useful Story Generator based on Sketch Premise Generator.

Here's a Hybrid Story Generator, taking starts from Story Generator and conflicts from Sketch Premise Generator (the best of both worlds maybe).
